## 🛠️ How to Use

1. **Clone the repo**
   ```bash
   git clone https://github.com/sakshamVerma08/ml-practice.git
   cd ml-practice
   ```
2. **Set up the environment** (recommended via `conda`):
   ```bash
   conda create -n ml-course python=3.10
   conda activate ml-course
   pip install -r requirements.txt
   ```
3. **Launch a notebook**
   ```bash
   jupyter notebook
   ```
   Open any `.ipynb` file and start experimenting.
